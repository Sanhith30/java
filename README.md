## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or above
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or terminal

### Running a Project

```bash
# Clone the repository
git clone https://github.com/Sanhith30/java.git
cd java

# Compile any file
javac "Atm.java"

# Run the compiled class
java Atm
```

> For files with spaces in their name, wrap the filename in quotes during compilation.

```bash
javac "Library Management System.java"
java "Library Management System"
```

## Project Highlights

### ATM System
Simulates a basic ATM with account authentication, balance inquiry, deposit, and withdrawal operations.

### E-Commerce Console Platform
Implements product listing, cart management, order placement, and basic billing logic in a console interface.

### Employee Management System
Handles employee records including add, update, delete, and display operations using OOP principles.

### Hotel Room Booking System
Manages room availability, booking confirmation, and check-in/check-out workflows.

### Online Examination System
Presents multiple-choice questions, tracks responses, calculates scores, and displays results.

### Payroll System
Processes salary calculations based on employee designations, deductions, and allowances.

### Smart Home Device Controller
Simulates controlling smart home devices such as lights, fans, and locks through a menu-driven interface.
